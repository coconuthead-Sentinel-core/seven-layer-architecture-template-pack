## How to use

Each template is a blank scaffold. Copy the relevant template into your project's `docs/` folder and populate per the structure provided. Both packs are designed to be cross-referenced — the Phase Pack governs lifecycle, the A2A/MCP Pack governs agent-protocol surfaces.
